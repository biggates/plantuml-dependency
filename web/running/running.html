Running

To launch class diagram text generation using the PlantUML format, you should have Java installed on your machine.

Here are the JVMs which have been successfully tested :
  • java version "1.6.0_18" OpenJDK Runtime Environment (IcedTea6 1.8.2) (6b18-1.8.2-4ubuntu2) OpenJDK Server VM (build 16.0-b13, mixed mode)
  • java version "1.6.0_22" Java(TM) SE Runtime Environment (build 1.6.0_22-b04) Java HotSpot(TM) Server VM (build 17.1-b03, mixed mode)

To run PlantUML Dependency class diagram generation using the generated text description, please refer to the PlantUML documentation.

You can then download it from the download page and save it on your local disk.
