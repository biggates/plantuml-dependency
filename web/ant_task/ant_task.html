PlantUML Dependency Ant task

You can run PlantUML Dependency using the following simplest Ant example :

	<project name="OwnTaskExample" default="main" basedir=".">
		<!-- task definition -->
		<taskdef name="plantuml-dependency" classname="net.sourceforge.plantuml.dependency.main.ant.PlantUMLDependencyProgramTask" classpath="/home/graffity/workspace/plantuml-dependency/target/plantuml-dependency-1.1.0-jar-with-dependencies.jar" />

		<!-- process all java files within the current directory to output "/home/plantuml.txt" file -->
		<target name="main">
		<plantuml-dependency output="/home/plantuml.txt" />
		</target>
	</project>

This Ant task processes all .java of the current directory and output a plantuml.txt text file.

Input Directory

You can also specify a input base directory to look for the .java files, using baseDir tag.

	<project name="OwnTaskExample" default="main" basedir=".">
		<!-- task definition -->
		<taskdef name="plantuml-dependency" classname="net.sourceforge.plantuml.dependency.main.ant.PlantUMLDependencyProgramTask" classpath="/home/graffity/workspace/plantuml-dependency/target/plantuml-dependency-1.1.0-jar-with-dependencies.jar" />

		<!-- process "/home" directory to output "/home/plantuml.txt" file -->
		<target name="main">
			<plantuml-dependency output="/home/plantuml.txt" baseDir="/home" />
		</target>
	</project>

This Ant task processes all .java of the /home directory and output a plantuml.txt text file.

Includes and excludes files

You can also specify a set of files to include or exclude when browsing the input base directory, using includes and excludes tags.

	<project name="OwnTaskExample" default="main" basedir=".">
		<!-- task definition -->
		<taskdef name="plantuml-dependency" classname="net.sourceforge.plantuml.dependency.main.ant.PlantUMLDependencyProgramTask" classpath="/home/graffity/workspace/plantuml-dependency/target/plantuml-dependency-1.1.0-jar-with-dependencies.jar" />

		<!-- process "/home" to output "/home/plantuml.txt" file -->
		<target name="main">
			<plantuml-dependency output="/home/plantuml.txt" includes="**/*Test*.java" excludes="**/PlantUMLDependencyProgram.java" baseDir="/home" />
		</target>
	</project>

This Ant task processes all *Test*.java (except PlantUMLDependencyProgram.java) of the /home directory and output a plantuml.txt text file.

Other parameters

Parameter Default Description
verboseLevel 800 Specifies the verbose level. The argument may consist of either a level name or an integer value. Classical values are : "SEVERE":1000, "WARNING":900, "INFO":800, "CONFIG":700, "FINE":500, "FINER":400, "FINEST":300.
Note that options are just ignored if they have empty values. For instance, the verboseLevel option doesn't behave the same as the command line.
